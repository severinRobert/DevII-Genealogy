# DevII-Genealogy
## What is it ?
This is a project for a python course. The goal is to create an app that will centralise several genealogy platform like Geneanet, FamilySearch and Heredis. 

## Run project
create a secrets.yml with this content : 
```
geneanet:
  cookie: ""
  sourcename: ""
familysearch:
  cookie: ""
  
```
